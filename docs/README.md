## 🎯 Panduan Cepat

### Untuk Admin
1. **Login** → Email & Password
2. **Input Setoran** → Menu Setoran → Bulk Input
3. **Kelola Anggota** → Menu Anggota
4. **Cek Stok** → Menu Stok
5. **Export Laporan** → Menu Laporan

### Untuk Anggota
1. **Akses Portal** → Scan QR atau buka link
2. **Login** → Masukkan 6-digit PIN
3. **Lihat Riwayat** → Cek setoran per bulan
4. **Download PDF** → Simpan sebagai bukti
